Detailed Description

A class describing a Matrix object. Matrices are array of size (n,m) whre n and m are integer greater or equal than 1. Notice that in general a matrix is not square. This class is intended to be used a simply as with math formulae.

Warning
Indices are 1-based (use them as you would do in math formulae)
