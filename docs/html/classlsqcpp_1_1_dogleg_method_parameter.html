◆ setAcceptanceFitness()

template<typename _Scalar >
void lsqcpp::DoglegMethodParameter< _Scalar >::setAcceptanceFitness ( const Scalar  fitness)
inline

Set the minimum fitness value at which a model is accepted. The model fitness is computed as follows:

fitness = f(xval) - f(xval + step) / m(0) - m(step)

Where f(x) is the objective error function and m(x) is the model function describe by the trust region method.
