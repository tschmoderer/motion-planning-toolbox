Detailed Description

template<typename _Scalar, int _Inputs, int _Outputs, typename _Objective, typename _StepMethod, typename _RefineMethod, typename _FiniteDifferencesMethod>
class lsqcpp::LeastSquaresAlgorithm< _Scalar, _Inputs, _Outputs, _Objective, _StepMethod, _RefineMethod, _FiniteDifferencesMethod >

Core class for least squares algorithms. It implements the whole optimization strategy except the step calculation and its refinement.
